# India Clean Energy Access Dashboard (SDG-7)

This project presents an interactive Power BI dashboard analyzing India's progress toward SDG-7: Affordable and Clean Energy using datasets from the National Data and Analytics Platform (NDAP) by NITI Aayog.

## Key Insights
- State-wise electrification rate
- LPG access rate by state
- Geographical distribution of energy access
- Comparison of electrification vs LPG access

## Tool Used
Microsoft Power BI Desktop

## Dataset Source
National Data and Analytics Platform (NDAP) – NITI Aayog
